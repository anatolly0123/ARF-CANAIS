## Executar Localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
   `npm install`
2. Configure a `GEMINI_API_KEY` no arquivo [.env.local](.env.local) com a sua chave de API do Gemini
3. Inicie o app:
   `npm run dev`
