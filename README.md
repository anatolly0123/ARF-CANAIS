# Execute e publique seu app do AI Studio

Este repositório contém tudo o que você precisa para rodar seu app localmente.

Veja seu app no AI Studio: https://ai.studio/apps/43e1012a-aada-4d4b-8e90-19b98f5801f9

## Executar Localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
   `npm install`
2. Configure a `GEMINI_API_KEY` no arquivo [.env.local](.env.local) com a sua chave de API do Gemini
3. Inicie o app:
   `npm run dev`
